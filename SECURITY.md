# Security Policy

## Supported Versions

The `main` branch and the latest tagged release are supported.

## Reporting a Vulnerability

Please report vulnerabilities privately to the maintainer before public disclosure.

Contact: florin@badita.org

Do not attach real evidence, credentials, malware, or private case material to public issues.

## Evidence Handling

Forensics in Tab is designed so evidence files stay in the investigator's browser. The app must not upload evidence bytes to any backend service.
